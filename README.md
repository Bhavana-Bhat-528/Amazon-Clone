# Amazon Clone

A frontend clone of the Amazon homepage built using HTML and CSS.

##  About the Project

This project is a recreation of the Amazon website interface created while learning frontend development. 
It focuses on building layouts, styling components, and improving CSS skills.

##  Features

- Amazon-style navigation bar
- Search bar UI
- Hero section
- Product/category cards
- Footer section
- Layout created using CSS Flexbox

##  Technologies Used

- HTML5
- CSS3
